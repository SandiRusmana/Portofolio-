# Proyek Portofolio Web Statis - Sandi Rusmana

Proyek ini adalah halaman portofolio personal yang dibuat untuk menampilkan informasi diri, keahlian, dan proyek yang telah saya kerjakan.

## 🚀 Fitur Halaman Utama
- **Hero Section:** Perkenalan singkat profil diri secara profesional.
- **Responsive Navigation:** Desain menu header yang ramah pengguna dan mendukung tampilan mobile.
- **Social Links:** Tautan langsung ke GitHub dan kontak WhatsApp.

## 🛠️ Teknologi yang Digunakan
- **HTML5:** Untuk menyusun struktur konten halaman web.
- **CSS3:** Untuk memberikan desain, tata letak modern (Flexbox), dan animasi interaktif.
- **Boxicons:** Untuk integrasi font icon (GitHub, WhatsApp, Instagram).
